# Breast Cancer Analysis
This project aims to develop and evaluate two classification models—Logistic Regression and Single Neuron Neural Network—to predict whether a breast mass is Malignant (cancerous, Class 0) or Benign (non-cancerous, Class 1). The goal is to identify the most effective model for accurately diagnosing the disease based on clinical measurements.

-python

-scikit-learn

-cv2
